# WordPress.org Plugin Assets

This directory contains assets for the WordPress.org plugin directory listing.

## Required Files

Place these images in this directory:

### Icon (Required)
- **icon-128x128.png** - Plugin icon for search results (128x128px)
- **icon-256x256.png** - Retina version (256x256px) - Optional but recommended

### Banner (Recommended)
- **banner-772x250.png** - Header banner for plugin page (772x250px)
- **banner-1544x500.png** - Retina version (1544x500px) - Optional

### Screenshots (Optional)
- **screenshot-1.png** - First screenshot (1280x720px recommended)
- **screenshot-2.png** - Additional screenshots

## Design Ideas

For the Raptorize It plugin, consider:
- Icon: A raptor silhouette or head
- Banner: A dramatic raptor scene or Jurassic Park-inspired design
- Screenshots: The raptor animation in action, Gutenberg block interface

## Deployment

The GitHub Action automatically uploads files from this directory to the WordPress.org SVN `assets` folder when you publish a release.
